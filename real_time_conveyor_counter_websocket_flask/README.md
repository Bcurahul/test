### Steps on how to run:

1.  Create virtual environment
```
python -m venv .venv
```

2.  Activate the virtual environment
```
source .venv\bin\activate
```
3.  Install the dependencies
```
pip install -r requirements.txt
```
4.  Run the project
```
python conveyor_counter_app.py
```
